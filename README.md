
# TurnoBanco Frontend
Aplicación móvil desarrollada en Flutter para la gestión de turnos bancarios.

## Descripción
Este proyecto implementa el MVP del caso TurnoFácil adaptado a un banco.
Incluye:
- Lista de servicios
- Detalle del servicio
- Solicitud de turno

## Cómo ejecutar
1. Tener Flutter instalado.
2. Abrir un emulador Android o conectar un dispositivo.
3. Ejecutar en terminal:
flutter run

## Tecnologías usadas
- Flutter
- Dart

## Flujo principal
Lista de servicios -> Detalle del servicio -> Solicitar turno
